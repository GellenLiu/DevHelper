## 🚀 快速开始步骤

### 1. 安装（2 分钟）
```
1. 打开 chrome://extensions/
2. 启用开发者模式
3. 点击"加载未封装的扩展程序"
4. 选择 apiProxy 文件夹
✓ 完成！
```

### 2. 配置规则（2 分钟）
```
1. 点击插件图标
2. 点击"+ 添加新规则"
3. 填写源 URL、目标 URL
4. 点击"保存规则"
✓ 完成！
```

### 3. 测试（1 分钟）
```
1. 打开浏览器控制台
2. 执行 fetch 请求
3. 在插件日志中查看
✓ 完成！
```
