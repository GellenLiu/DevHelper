# Web Performance Tester

🚀 **精准测试网页性能，深度分析用户体验**

## 🌟 核心亮点

### 🎯 自定义RTTI测试
**RTTI (Render Time of Targeted Element)** - 插件最大亮点！
- 支持选择页面任意元素进行精准渲染时间测试
- 可视化元素选择器，直观易用
- 实时反馈元素渲染性能，帮助定位性能瓶颈

### 📊 全面性能指标
- **基础指标**：加载时间、FCP、TTI、LCP、CLS
- **网络指标**：DNS解析时间、TCP连接时间、TTFB、网络总耗时
- **用户体验指标**：白屏时间、TBT（总阻塞时间）、长任务数
- **流畅度指标**：FPS（帧率）

### 🔄 多轮测试取平均值
- 支持1-10次测试，自动计算平均值
- 可调整测试间隔（500-5000ms）
- 测试结果更精准，避免单次测试偏差

### 🎨 直观可视化结果
- 性能数据分类显示，一目了然
- 颜色编码性能等级：绿色（优秀）、黄色（警告）、红色（较差）
- 支持查看详细性能报告

## 📋 功能特性

## 🚀 快速开始

### 1. 安装插件
- 打开浏览器扩展管理页面
- 开启"开发者模式"
- 点击"加载已解压的扩展程序"
- 选择本项目文件夹

### 2. 使用步骤

1. **打开插件**：点击浏览器右上角插件图标
2. **选择元素**（可选，用于RTTI测试）：
   - 点击元素选择器按钮
   - 鼠标悬停在页面元素上，选择目标元素
3. **配置测试**：
   - 设置测试次数（1-10次）
   - 调整测试间隔（500-5000ms）
4. **开始测试**：点击"开始检测"按钮
5. **查看结果**：
   - 等待测试完成
   - 查看各项性能指标平均值
   - 点击"更多"查看详细报告

## 📈 性能指标说明

| 指标 | 描述 | 优秀阈值 | 警告阈值 |
|------|------|----------|----------|
| **加载时间** | 页面完全加载所需时间 | < 2000ms | < 4000ms |
| **FCP** | 首次内容绘制时间 | < 1800ms | < 3000ms |
| **TTI** | 可交互时间 | < 3800ms | < 7300ms |
| **RTTI** | 目标元素渲染时间 | < 2000ms | < 4000ms |
| **LCP** | 最大内容绘制时间 | < 2500ms | < 4000ms |
| **CLS** | 累积布局偏移 | < 0.1 | < 0.25 |

## 🔧 技术实现

- **Manifest V3**：使用最新的Chrome扩展开发规范
- **Performance API**：利用浏览器原生性能API收集数据
- **MutationObserver**：实时监控DOM变化，计算RTTI
- **Storage API**：保存配置和测试结果
- **Content Script**：注入页面脚本，收集性能数据
