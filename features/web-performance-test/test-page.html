性能测试页面

用于测试浏览器性能指标的综合页面

页面介绍

这个页面包含多种类型的内容，用于测试浏览器的各项性能指标。包括文本、图片、视频、动态内容和布局变化等。

您可以使用性能测试插件来测量以下指标：

  • 加载时间 - 页面完全加载所需的时间
  • FCP (First Contentful Paint) - 首次内容绘制时间
  • LCP (Largest Contentful Paint) - 最大内容绘制时间
  • CLS (Cumulative Layout Shift) - 累积布局偏移
  • TTI (Time to Interactive) - 可交互时间
  • RTTI (Resource Time to Interactive) - 资源可交互时间
