# Chrome Bookmark Reader

## 功能描述

该技能可以：
- 自动读取Chrome浏览器的收藏书签
- 以结构化格式返回书签数据给大模型
- 支持书签的快速检索
- 支持模糊搜索功能
- 支持按文件夹查看书签
- 直接响应书签查询请求

## 如何使用

当用户需要：
- 分析Chrome浏览器的收藏书签
- 快速检索特定的书签
- 搜索与特定关键词相关的书签
- 了解书签的组织结构
- 直接查询特定主题的书签

## 技术实现

该技能通过以下步骤实现：
1. 读取Chrome浏览器的书签文件
2. 解析书签数据结构
3. 提供结构化的书签信息
4. 实现书签的搜索和过滤功能
5. 直接响应用户的书签查询请求

## 示例用法

### 示例1：获取所有书签
用户："请帮我分析一下我的Chrome书签"

### 示例2：搜索特定书签
用户："我想找到关于JavaScript的书签"

### 示例3：按文件夹查看书签
用户："请查看我的'技术学习'文件夹中的书签"

### 示例4：直接查询主题书签
用户："查询内容管理的书签"
用户："查找与编程相关的书签"

## 注意事项

- 该技能仅读取书签数据，不会修改任何书签
- 书签分析基于本地Chrome浏览器数据
- 搜索功能支持模糊匹配，可返回相关度较高的结果
- 无需创建额外脚本，直接通过自然语言查询即可使用或内置工具执行bookmark_reader.js的脚本
- 如果创建了执行脚本，任务完成后请删除该脚本
